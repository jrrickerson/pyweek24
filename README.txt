	uniseg==0.7.1


Running the Game
----------------

Open a terminal / console and "cd" to the pyweek24 directory and run:

 	python3 -m behind.rungame


How to Play the Game
--------------------

    left, right arrow keys or 'a', 'd' to move
    space to jump
    ESC to quit
